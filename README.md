# MyForum
這是一個用ASP.NET撰寫的線上論壇
## 開發環境
### 資訊
Visual Studio 2017 Community
.NET Framwork 4.6.1
### 說明
本網路應用程式（Web Application）係利用微軟開發之Visual Studio並基於使用ASP.NET進行開發。ASP.NET是由微軟在 .NET Framework框架中所提供，開發Web應用程式的類別庫。
